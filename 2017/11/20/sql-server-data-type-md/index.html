SQL Server数据类型

文本数据类型

  • char 固定长度的非Unicode字符
  • varchar 可变长度的非Unicode字符
  • nchar 固定长度的Unicode支付
  • nvarchar 可变长度的Unicode支付
  • text 储存长文本信息
  • ntext 储存可变长度的文本信息

    日期和时间

  • datetime 储存时间和日期

    数字数据

    整数

  • int smallint tinyint bigint

    小数 可以储存小数，并且可以指定数据总宽度和小数位数

  • float decimal

    货比数据类型

  • money 可以用来存放货比类型

    Bit数据类型

  • bit 储存布尔数据类型 True/False

    二进制数据类型

  • image 可以用来储存图像
